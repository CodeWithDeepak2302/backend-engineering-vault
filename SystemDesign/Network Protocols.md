

**A Protocol** is the strict set of rules that governs how data physically moves from point A to point B across a network. It dictates packet structures, handshakes, and error handling. It’s the highway system and the traffic laws.

Protocols define
1. How connections are established
2. How data is framed
3. How errors are handled
4. How bytes move over the wire
